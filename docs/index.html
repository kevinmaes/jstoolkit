toKeys

Get keys from an array or an object

toKeys
Parameters
keys ((Array<string> | object)) An array or object whose elements or keys are key values.
Returns
Array<string>: An array of key values (strings).

keyMirror

keyMirror
Parameters
keys ((Array<string> | object)) An array or object whose elements or keys are key values.
Returns
object: An object whose key/value pairs match the keys passed in.
Example
const keys = ['one', 'two', 'three']
const mirror = keyMirror(keys)
console.log(mirror) // { one: 'one', two: 'two', three: 'three' }
const keys = { one: null, two: null, three: null }
const mirror = keyMirror(keys)
console.log(mirror) // { one: 'one', two: 'two', three: 'three' }
